Выборы главного админа в dshindov_chat

Выборы проходят по системе instant-runoff voting. В своём бюллетене вам нужно будет выбрать несколько кандидатов и упорядочить их по убыванию приоритета. Изначально кандидаты ранжируются по количеству бюллетеней, в которых их выбрали первыми. Если же по этим результатам не вышло определить победителя, то кандидат с наименьшим количеством голосов выбывает, и голоса идут тем кандидатам, которые были выбраны в бюллетенях вторыми. Так продолжается, пока у кого-то из кандидатов не наберётся более 50% голосов.
